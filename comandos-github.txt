  COMANDOS PARA CONECTAR AO GITHUB E FAZER PRIMEIRO PUSH
═══════════════════════════════════════════════════════════════════

⚠️ IMPORTANTE: Substitua "SEU-USUARIO" pelo seu nome de usuário do GitHub!

═══════════════════════════════════════════════════════════════════
PASSO 2: CONECTAR REPOSITÓRIO LOCAL AO GITHUB
═══════════════════════════════════════════════════════════════════

Execute estes comandos no PowerShell (na pasta do projeto):

1️⃣ Conectar ao GitHub:
─────────────────────────
git remote add origin https://github.com/SEU-USUARIO/landing-page-vet.git

Substitua SEU-USUARIO pelo seu usuário real!
Exemplo: https://github.com/joaosilva/landing-page-vet.git


2️⃣ Renomear branch para main:
─────────────────────────────
git branch -M main


3️⃣ Enviar código para o GitHub:
─────────────────────────────────
git push -u origin main


═══════════════════════════════════════════════════════════════════
AUTENTICAÇÃO (Primeira vez)
═══════════════════════════════════════════════════════════════════

Quando executar "git push", o GitHub vai pedir:

Username: [seu email ou nome de usuário]
Password: [USE UM TOKEN, NÃO A SENHA DA CONTA]


COMO CRIAR TOKEN:
─────────────────
1. Acesse: https://github.com/settings/tokens
2. Clique em "Generate new token (classic)"
3. Nome: "Landing Page Deploy"
4. Marque: ☑ repo (Full control of private repositories)
5. Clique em "Generate token"
6. COPIE O TOKEN (não vai aparecer novamente!)
7. Cole como senha quando o Git pedir
